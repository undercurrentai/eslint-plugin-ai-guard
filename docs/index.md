# @undercurrent/eslint-plugin-ai-guard

Framework-deep, AI-risk-focused ESLint plugin for JS/TS application code.

- Adoption-first `recommended` preset
- `strict` preset for full enforcement
- `security` preset for AppSec pipelines
- `framework` preset — auth/authz/webhook signatures across Express, Fastify, Hono, NestJS, Next.js
- `compat` preset — disables deprecated rules cleanly
